Et AI-drevet webværktøj til automatisk vurdering af praktikrapporter, med frontend via GitHub Pages og backend via Render.

## Projektbeskrivelse

AI Vurderings-Tool giver brugeren mulighed for at uploade eller indsætte en praktikrapport og få den vurderet automatisk ved hjælp af OpenAI's API. Systemet analyserer rapportens indhold og returnerer struktureret feedback med styrker, svagheder og en samlet bedømmelse.

Projektet er bygget med en opdelt arkitektur: frontenden er en statisk React-applikation hostet på GitHub Pages, mens backenden er en separat REST API-service deployet på Render.

## Funktioner

- Automatisk AI-vurdering af praktikrapporter
- Struktureret feedback med styrker og forbedringsområder
- Simpel og overskuelig brugerflade
- Opdelt frontend/backend-arkitektur

## Tech stack

- React (frontend)
- GitHub Pages (frontend hosting)
- Node.js / Express (backend)
- Render (backend hosting)
- OpenAI API

## Hvad jeg arbejdede med

Jeg designede og implementerede hele applikationen – fra React-frontenden til Express-backenden med OpenAI-integration. Projektet gav erfaring med API-nøglehåndtering på serversiden, CORS-opsætning på tværs af domæner og cloud-deployment af både statiske og dynamiske tjenester.
