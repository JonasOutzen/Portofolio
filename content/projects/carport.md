A full-stack semester project developed for Johannes Fog, focused on improving the process of designing, pricing, and managing custom carport offers.

## GitHub

[View the repository](https://github.com/MarcusPFF/Carport-Fog)

## Project overview

This application allows customers to configure a carport, enter their information, and request a non-binding offer. The system generates a price estimate and lets the customer view a dynamic technical drawing of the selected carport. Sellers can then manage incoming offers, update dimensions and pricing, and continue the sales process through an internal dashboard. :contentReference[oaicite:5]{index=5}

If the customer accepts the offer, the system can generate a material list based on the final agreement. :contentReference[oaicite:6]{index=6}
