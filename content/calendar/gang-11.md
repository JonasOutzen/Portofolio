**Gang:** 11 (22/5)

**Tema:** Projektarbejde + code review

**Indhold:** Projektarbejde med fælles code review. Vi undersøger, om brugen af kodeagenter og spec-driven development faktisk hjælper med at holde kvaliteten og processen oppe.

## Hvad vi gjorde

Vi holdt fælles code review på tværs af grupper, hvor vi kiggede på hinandens kode og gav feedback. Diskussionerne handlede om:
- Kodestruktur og læsbarhed
- Brug af kodeagenter i processen – hvad de hjalp med, og hvad de introducerede af problemer
- Spec-driven development i praksis – fulgte vi vores specifikationer?

Det gav et godt indblik i, hvordan andres projekter er bygget op, og hvad man kan gøre anderledes i sin egen løsning.

## Output / portfolio

Dokumentation af udviklingsprocessen – herunder refleksioner over code review-sessionen og brugen af AI-assisterede udviklingsmetoder.
